Nazwa Typ Opis Opcje
STP:Id LONG Identyfikator
 SQL_IDENTITY
STP:Kod CSTRING(41) Kod typu przeglądu
STP:Nazwa CSTRING(101) Nazwa typu przeglądu
STP:Opis CSTRING(256) Opis typu przeglądu
STP:Aktywny BYTE Oznaczenie czy typ przeglądu jest aktywny
(czy można go wybrać na plan przeglądu i dodać do grupy urządzeń/urządzenia). 0 - nieaktywny, 1 - aktywny
STP:Domyslny BYTE Oznaczenie, czy typ przeglądu wprowadzać jako domyślny na plan przeglądu.
0 – niedomyślny, 1 – domyślny.
STP:Cykliczny BYTE Czy typ przeglądu powtarzalny.
0 – niepowtarzalny, 1 – cykliczny.
STP:CzasCyklu LONG Powtarzaj co (czas cyklu)
STP:CzasCykluJedn BYTE Powtarzaj co (jednostka czasu cyklu)
0 - dzień, 1 - tydzień, 2 - miesiąc, 3 - rok
STP:Terminarz BYTE Zapis w terminarzu
0 - nie generuj zapisu w terminarzu 1 - generuj zapis bez przypomnienia 2 - generuj zapis z przypomnieniem
STP:CzasPrzypomnienia LONG Czas przypomnienia
STP:CzasPrzypomnieniaJedn BYTE Jednostka czasu przypomnienia
0 - sekundy, 1 - minuty, 2 - godziny, 3 - dni, 4 - tygodnie
STP:AutoNastepnyPrzeglad BYTE Generuj automatycznie następny przegląd na podstawie daty z AutoPrzegladZrodlo
STP:AutoPrzegladZrodlo BYTE 0-daty wystawienia ZSR,1-realizacji ZSR,2-poprzedniego przeglądu,3-przyjęcia/odrzucenia, 4-potwierdzenia
