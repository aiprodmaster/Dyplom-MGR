Wskaźnik edycji dokumentu. Jeżeli dokument jest w edycji, w ITN_Aktywny wpisywane jest SES_SesjaId bieżącej sesji. Jeśli dokument nie jest aktualnie w edycji ITN_Aktywny=0.
ITN:InwBlokujeModyfikacje BYTE Inwentaryzacja blokuje modyfikacje
0 - Wszystkich środków trwałych, 1 - Tylko inwentaryzowanych
ITN:GenerujArkuszeWg BYTE Generowanie arkuszy według
0 - Środków trwałych, 1 - Miejsc użytkowania, 2 - Osób mat. odpowiedzialnych 3 - Miejsc oraz osób mat. odpowiedzialnych
ITN:RodzajOdczytow BYTE Rodzaj odczytów
0 - Porównawcze, 1 - Przyrostowe
ITN:GenerowanieDokData BYTE Generowanie dokumentów z datą
0 - Zakończenia inwentaryzacji, 1 - Bieżącą
