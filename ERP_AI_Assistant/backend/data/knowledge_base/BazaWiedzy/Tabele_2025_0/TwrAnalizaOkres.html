TwrAnalizaOkres(MSSQL/PostgreSQL)

Dane o okresie analizy kategorii ABC/XYZ towarów

Prefiks: TAO
Zadeklarowana w: CDN_GLB3
Opcje: NO_FILTER

  Kolumny    Indeksy    Relacje  
  Clarion    MSSQL    PostgreSQL  
Nazwa Typ Opis Opcje
TAO:Id LONG Identyfikator okresu analizy
 SQL_IDENTITY
TAO:DataAnalizy LONG Data rozpoczęcia analizy
Data rozpoczęcia analizy ABC/XYZ w formacie TimeStamp
TAO:OkresRok SHORT Rok dla jakiego jest przeprowadzana analiza
0 - dla opcji zakres dat
TAO:OkresTyp BYTE Typ podziału roku, gdy analiza za jakiś okres roku
0 analiza za cały rok, 1 analiza za miesciąc, 2 analiza za kwartał, 3 analiza za półrocze, 4 analiza dla zakresu dat
TAO:OkresWartosc BYTE Wskazanie na konkretny przedział czasu dla dangeo typu
np OkresRok = 2009 i OkresTyp = 1 i OkresWartosc = 2 to analiza za luty 2009, OkresRok = 2008 i OkresTyp = 2 i OkresWartosc = 4 to analiza za czwarty kwartał 2008, OkresRok = 2010 i OkresTyp = 0 i OkresWartosc = 0 to anazliza z całego roku 2010, OkresRok = 0 i OkresTyp = 4 i OkresWartosc = 0 i OkresDataOd = 77154 i OkresDataDo = 77194 to analiza dla okresu od 2012-03-25 do 2012-05-04
TAO:ABCDataOd LONG Data rozpoczęcia okresu analizy ABC dla opcji zakres dat
TAO:ABCDataDo LONG Data zakończenia okresu analizy ABC dla opcji zakres dat
TAO:XYZDataOd LONG Data rozpoczęcia okresu analizy XYZ dla opcji zakres dat
TAO:XYZDataDo LONG Data zakończenia okresu analizy XYZ dla opcji zakres dat
TAO:Wykorzystana BYTE Czy wszystkie elementy analizy zostały dodane już do raportów ABC/XYZ
0-Pozostają elementy analizy niedodane do raportów ABC/XYZ 1-Wszystkie elementy są już dodane do raportów ABC/XYZ
