TAP:Status BYTE Status towaru
Określa czy towar jest: zapowiedzią, dostępny (w regularnej sprzedaży), dostępny za zamówienie, niedostępny - możliwe wartości: 0, 1, 2, 3.
TAP:Dostepnosc BYTE Określa czas w jakim towar będzie dostępny w sklepie
- możliwe wartości: 0, 1, 2, 3, 4, 5, 6, 7.
TAP:KontrolaStanow BYTE Określa czy kontrolowane będą stany towarowe dla towaru
możliwe wartości: 0, 1
TAP:PodlegaRabatowaniu BYTE Określa czy towar podlega rabatowaniu
możliwe wartości: 0, 1.
TAP:IndywKosztDst BYTE Indywidualny koszt dostawy
możliwe wartości: 0, 1.
TAP:Flagi LONG Flagi bitowe jakie można przypisać do towaru
