TrE:TrnTStamp LONG Fizyczny czas utworzenia transakcji
Fizyczny czas utworzenia transakcji, w postaci TimeStamp. Uwaga, jeśli data transakcji jest zgodna z zegarem systemowym, wówczas jest tu wpisywany czas systemowy. Jeśli data sprzedaży (zakupu) jest większa od daty systemowej, wówczas wpisywana jest godz. 23:59:59 z dnia sprzedaży (zakupu). Jeśli data sprzedaży (zakupu) jest mniejsza od daty systemowej, wówczas wpisywana jest godz. 0:00:01 z dnia sprzedaży (zakupu).
