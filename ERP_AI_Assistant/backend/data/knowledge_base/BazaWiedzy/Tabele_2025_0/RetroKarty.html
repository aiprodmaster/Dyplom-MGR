Nazwa Typ Opis Opcje
RtK:Id LONG Identyfikator rekordu
RtK:Nazwa CSTRING(201) Nazwa rabatu retrospektywnego
RtK:Typ BYTE Typ rabatu retrospektywnego
0 - zakup, 1 - sprzedaż
RtK:DataOd LONG Data początkowa
RtK:DataDo LONG Data końcowa
RtK:TypDaty BYTE Typ daty
0 - wystawienia, 1 - zakupu, 2 - sprzedaży.
RtK:Stan BYTE Stan
0 - Niepotwierdzony, 1 - Potwierdzony, 3 - Zamknięty
RtK:OpeUNumer LONG Operator, który utworzył nagłówek

RtK:OpeMNumer LONG Operator, który jako ostatni modyfikował nagłówek
RtK:CzasModyfikacji LONG Czas ostatniej modyfikacji
RtK:OkresyRozliczeniowe BYTE Okresy rozliczeniowe
0 - nie, 1 - tak
RtK:OkresIlosc LONG Okresy rozliczeniowe - ilość
RtK:OkresTyp BYTE Okresy rozliczeniowe - typ
0 - Dzień, 1 - Miesiąc, 2 - Kwartał, 3 - Rok
RtK:Aktywny LONG Numer bieżącej sesji
RtK:TypProgu BYTE Typ progu
0 - wartościowy, 1 - ilościowy, 2 - ilość odbiorców.
RtK:PodstawaRabatu BYTE Podstawa rabatu
0 - zakup u dostawcy, 1 - Sprzedaż towarów dostawcy
RtK:TypKalkulacji LONG
0 - wartość zakupu, 1 - katalogowa wartość zakupu, 2 - wartość sprzedaży, 3 - cennik,
RtK:Opis CSTRING(1025) Opis
RtK:SQL CSTRING(4097) Dodatkowy warunek SQL zdefiniowany dla bieżącego retro bonusu
RtK:TypID LONG Typ rabatu retrospektywnego - Zgodne z SlwId
RtK:RodzajCeny SHORT Rodzaj ceny
RtK:WymaganaZaplata BYTE Wymagana zapłata
RtK:TypTerminu BYTE 0 - Planowana ilość dni, 1 - Ilość dni zwłoki
